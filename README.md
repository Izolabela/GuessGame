# GuessGame
Simple Python Guess Game with PyAutoGUI

# Clone
```bash
git clone https://github.com/MemeSec/GuessGame && cd guessGame 
```

# Requirements
+ Tkinter (Pacman)
```bash
sudo pacman -S tk
```
+ Tkinter (apt)
```bash
sudo apt install tk
```
+ PyAutoGui
```bash
python3 -m pip install PyAutoGUI
```

# Reset Score
To reset the score just run **Reset-Score.py**
```bash
python3 Reset-Score
```
